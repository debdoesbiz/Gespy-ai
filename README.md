## 🧠 How It Works

1. Enter a business niche
2. Gespy searches and collects raw data
3. AI filters and structures results
4. Export clean leads instantly

---

## 🛠️ Run Locally

**Prerequisites:** Node.js

1. Install dependencies

```bash
npm install
```

2. Set your API key in `.env.local`

```env
GEMINI_API_KEY=your_api_key_here
```

3. Run the app

```bash
npm run dev
```
